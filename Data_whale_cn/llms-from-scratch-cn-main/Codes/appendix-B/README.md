# 翻译统一表

主要避免一词多译，为学者带来困扰

## 不需要翻译的术语

| 英文          | 大意                |
|-------------|-------------------|
| token       | 词元                |
| Transformer | 一种网络架构            |
| GPT         | 仅编码器架构 chatgpt的底座 |
| llama       | 类gpt 模型           |
| Dropout     | 一种减轻过拟合的手段 随机失活参数 |
