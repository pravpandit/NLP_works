# एलएलएम-आधारित एप्लिकेशन विकसित करने के लिए लैंगचेन का उपयोग करना

एंड्रयू एनजी द्वारा जारी बड़े मॉडल विकास पर नया पाठ्यक्रम डेवलपर्स को एलएलएम-आधारित अनुप्रयोगों के निर्माण के लिए लैंगचेन फ्रेमवर्क के साथ चैटजीपीटी एपीआई का उपयोग करने के बारे में मार्गदर्शन करता है, और डेवलपर्स को लैंगचेन का उपयोग करने के बारे में कुछ सुझाव सीखने में मदद करता है, जिसमें शामिल हैं: मॉडल, संकेत और पार्सर्स , अनुप्रयोगों के लिए आवश्यक भंडारण, मॉडल श्रृंखलाओं का निर्माण, दस्तावेज़-आधारित प्रश्न-उत्तर प्रणाली, मूल्यांकन और एजेंट, आदि।
